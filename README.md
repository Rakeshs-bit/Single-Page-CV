# 🧾 Single Page HTML CV

This project is a clean and structured **single-page CV (Curriculum Vitae)** built using only HTML.  
It focuses on semantic structure, SEO basics, and readiness for future styling with CSS.

---

## 🌐 Live Project

👉 https://github.com/Rakeshs-bit/Single-Page-CV

---

## 🚀 Features

- Semantic HTML structure (header, section, footer)
- Clean single-page layout
- SEO meta tags included
- Open Graph (OG) tags for social sharing
- Favicon integration
- Minimal and professional design
